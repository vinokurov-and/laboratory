# Лабораторная работа 1: Разверни React-приложение на GitHub Pages

## Цели работы
1. Cоздать React + TypeScript-приложения с помощью сборщика Vite.
2. Настроить автоматическую публикацию статического сайта на Github Pages.

## Что нужно
- ПК с установленным Node.js версии 20+ и npm.
- Учетная запись на GitHub.
- Установленный Git.
- Базовые знания командной строки (cd, ls, git, npm) 

---

## Часть 1: Создание React-приложения с Vite + Typescript

1. Открой терминал (в любой директории).
2. Создай новый проект с помощью Vite:
   ```
   npm create vite@latest lab1-react-vite-gh-pages -- --template react-ts
   ```
    Команда:
   - npm create vite@latest — создаёт проект через официальный инструмент
   - --template react-ts — выбирает шаблон React + TypeScript
   - lab1-react-vite-gh-pages — имя проекта (можно заменить)
  
3. Перейди в папку проекта:
   ```
   cd lab1-react-vite-gh-pages
   ```
4. Установи зависимости:
```
npm install
```
 5.Запусти локальный сервер и убедись, что всё работает:
```
npm run dev
```
- Открой браузер → http://localhost:5173
- Ты должен увидеть логотип Vite

## Часть 2: Создание React-приложения с Vite + Typescript
1. Установи пакет gh-pages — он поможет загрузить сборку на GitHub:
   ```
   npm install gh-pages --save-dev
   ```

2. Открой файл package.json и добавь/измени поля:

```
{
  "name": "lab1-react-vite-gh-pages",
  "homepage": "https://<ваш-username>.github.io/lab1-react-vite-gh-pages",
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "preview": "vite preview",
    "predeploy": "npm run build",
    "deploy": "gh-pages -d dist"
  }
}
```

⚠️ ВАЖНО:
Замени <ваш-username> на свой логин на GitHub (например, alexdev)
Путь к сборке теперь — dist/, а не build/ (Vite использует dist по умолчанию)

3. Сохрани package.json.

## Часть 3: Создание репозитория на GitHub
1. **Вход в GitHub:**
   - Перейдите на [GitHub](https://github.com).
   - Войдите в свою учетную запись.
2. **Создание нового репозитория:**
   - Нажмите на кнопку "+" в правом верхнем углу и выберите "New repository".
   - Дайте название репозиторию (например, `my-nest-project`).
   - Сделайте его публичным или приватным по вашему усмотрению.
   - Нажмите на кнопку "Create repository".

## Часть 4: Загрузка проекта на GitHub

1. **Инициализация Git в проекте:**
   - В терминале в папке вашего проекта выполните:
     ```bash
     git init
     ```

2. **Добавление файлов:**
   - Добавьте файлы в индекс Git:
     ```bash
     git add .
     ```

3. **Создание коммита:**
   - Создайте коммит с сообщением:
     ```bash
     git commit -m "feat: init Vite + React + TypeScript project"
     ```

4. **Связывание локального репозитория с удалённым:**
   - Замените `USERNAME` на ваше имя пользователя и `my-nest-project` на имя вашего репозитория:
     ```bash
     git remote add origin https://github.com/USERNAME/my-nest-project.git
     ```

5. **Отправка изменений на GitHub:**
   - Выполните команду для загрузки текста в удалённый репозиторий:
     ```bash
     git branch -M main
     git push -u origin main
     ```

## Часть 5: Разверни сайт на GitHub Pages
1. Запусти деплой
```
npm run deploy
```
2. Подожди -> Github Pages обрабатывает изменения
3. Открой в браузере
```
https://<ваш-username>.github.io/lab1-react-vite-gh-pages
```
→ Ты должен увидеть тот же сайт, что и на localhost:5173 — но теперь он живёт в облаке!
